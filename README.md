# 語録チェッカー
日本で生活していると、淫夢語録は避けて通れないものです。しかし、公共の場でそれらの表現を使用すると、誤解を招いたり、周囲の人を不快にさせてしまう可能性があります。

本来であれば、何らかのコンテンツを公開する前に、関係者の中に淫夢を知っている人がいれば、問題になりそうな表現を事前に指摘して事故を防ぐことができます。ところがここで問題が発生します。それは、指摘した人が「淫夢を知っていると周りにバレてしまう」ことです。さらに、「指摘したら淫夢厨ってバレるな…」という自己保身から、結局指摘がされないままになってしまうケースも少なくありません。

そこで、このツールの出番です。発言や文章を提出・公開する前に、作成した本人が事前にこのツールでチェックするようお願いしておけば、問題になり得る語録や構文を自動で検出できます。あらかじめ確認する手順を取り入れておくことで、誰かが直接指摘する必要もなくなり、「指摘したら淫夢厨ってバレるな…」という状況自体を避けることができます。

また、このツールは「文章を校正するツール」などとして説明しておけば、自然な形でチェックの仕組みを導入することが可能です。
